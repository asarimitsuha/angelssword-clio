# Memorial Portraits

Drop character portrait images here as PNG files.

## Naming Convention

Filename = character name, lowercased, spaces replaced with underscores, special characters removed.

Examples:
- Voltaire → `voltaire.png`
- Wizard of Poop → `wizard_of_poop.png`
- Deryn Ennastis → `deryn_ennastis.png`
- V-I7 → `v-i7.png`
- Li Shui → `li_shui.png`
- Mime-chan → `mime-chan.png`
- Sicle Claw → `sicle_claw.png`

If no portrait exists for a character, nothing will be shown.
